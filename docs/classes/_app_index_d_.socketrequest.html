Static castResponse

castResponse: function
  • Defined in app/index.d.ts:408

Decorator for casting response with defined types.

example
class User extends SocketRequest {
    @SocketRequest.castResponse({
        id: Type.integer,
        name: Type.string,
    })
    get() {
        return this.execute('user.get');
    }
}
